# Jave-practice-

This repository contains Java programming exercises and practice problems that I complete while studying Software Engineering.
The goal of this repository is to strengthen my understanding of core Java concepts and to maintain consistent coding practice.

## Topics Covered

The exercises in this repository cover the following Java concepts:

* Methods
* Arrays
* ArrayLists
* Conditional Statements
* Loops
* Exception Handling
* Object-Oriented Programming (OOP)
* Inheritance

